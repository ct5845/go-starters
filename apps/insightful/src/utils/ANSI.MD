	## ANSI Escape Codes

### Formatting

| Format Name      | ANSI Code | Full ANSI Escape Sequence |
| :--------------- | :-------- | :------------------------ |
| Reset            | `0`       | `\033[0m`                 |
| Bold             | `1`       | `\033[1m`                 |
| Dim              | `2`       | `\033[2m`                 |
| Underline        | `4`       | `\033[4m`                 |
| Underline and Bold| `1;4`    | `\033[1;4m`               |
| Reverse          | `7`       | `\033[7m`                 |
| Hidden           | `8`       | `\033[8m`                 |
| Strikethrough    | `9`       | `\033[9m`                 |
